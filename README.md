NoteStudio is a lightweight, browser-based note-taking application built with HTML, CSS, and vanilla JavaScript. The project focuses on clean structure, usability, and maintainable code without relying on external frameworks.

It is designed as both a functional productivity tool and a portfolio project demonstrating strong frontend fundamentals.

---

## Features

- Create and manage notes in a simple interface  
- Component-based UI structure (cards, modals, tooltips, navigation)  
- Light and dark theme support  
- Clean and consistent design following a style guide  
- Organized and readable codebase  
- **Time-based greetings** (Good morning, Good afternoon, Good evening, Good night) depending on the current time  
- **Display today's date** in a user-friendly format  
- **Note timestamp support** showing how long ago a note was created (e.g., "Just now", "5 mins ago", "2 hours ago", "3 days ago")
